== Description ==

Use this plugin to add The Armadillo to your WordPress website.

If a user is logged into their account, we will automatically pass the user information to the Armadillo to allow you to see it inside your account.

If the logged-in user is a WordPress Administrator, we also disable The Armadillo to not track you and your admins while visiting the site.

*Note:* This plugin requires an active subscription to Canny Armadillo to work. You can subscribe to Canny Armadillo on our website [cannyarmadillo.com](https://cannyarmadillo.com)

## Install

1. Upload the plugin to your WordPress site through the WordPress admin.
2. Activate the plugin.
